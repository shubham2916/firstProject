Get project into your local machine.

How to do it?

#Git clone URLOfTheProject.

1. Make changes/work on the project

#cd into project folder

#cd PROJECTFOLDER

You will notice that, the project is defaulted to the default branch

This default branch could be 'Master' or any other branch that the author has set up as 'Default' in github.

Somthething like below:

projectfolder (Master)>

or

projectfolder (develop)>

NOTE: Never work on the default branch

projectfolder (develop)> git checkout -b YOURBRANCHNAME

projectfolder (YOURBRANCHNAME) >

open project in your IDE and make changes

projectfolder (YOURBRANCHNAME) > git status

check all your changes

shown in red

projectfolder (YOURBRANCHNAME) > git add .

. refers to all

git add specificFileName

The added files are 'Staged'

'Staged' means ready to be committed

projectfolder (YOURBRANCHNAME) > git commit -m "Your description of change"

projectfolder (YOURBRANCHNAME) > git push -u origin YOURBRANCHNAME

1. Create PUll Request

When we push branch you will give option to create a PULL Request

You need go to Gitbub.com/To/your/project

You will notice a notification pop up asking to create a PR from your pushed branch

Click that and select the branch you want your branch to be merged into.

When you a pull request, collaborators will be notified of the PR and they need to check the changes and approve/comment/discard the PR.

1. How to check a pull request

You just checkout to default branch

projectfolder (MyLocalBranch) > git pull

checkout the PR requested Branch (This is available on github on the PR screen)

projectfolder (MyLocalBranch) > git checkout PRRequestedBranch

#check all the changes

#You can approve/disapprove of the changes on github.com/PR
